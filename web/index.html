Ultimate database

Ultimate Debian Database (UDD) gathers various Debian data in the same SQL database. It allows users to easily access and combine all these data. Read more on the ultimate database wiki page. Here are some UDD-based web services:

  • Bugs Search: multi-criteria search engine for bugs
  • Maintainer Dashboard: information about teams and maintainers' packages
  • Bugs Usertags: search for usertag on bugs
  • Sponsors Stats gives some statistics about who is sponsoring uploads to Debian
  • Bapase allows to search for "interesting" packages using various criterias
