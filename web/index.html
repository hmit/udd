Ultimate Debian Database (UDD)

General information

Ultimate Debian Database (UDD) gathers a lot of data about various aspects of Debian in a single SQL database. It allows users to easily access and combine all this data.

You can have a look at the UDD Schema.

More information about UDD are collected on the Debian Wiki at wiki.debian.org/UltimateDebianDatabase.

Examples

To give an idea what results UDD can deliver, here are some example queries: (source code)

  • Packages in unstable, but not in testing, sorted by popcon installs
  • Sources in unstable, but not in testing, sorted by maximum popcon installs
  • Sources in Ubuntu, but not in Debian, sorted by maximum popcon installs
  • Number of bugs per package
  • Number of RC bugs per package
  • Packages which are "good" candidates for removal from testing
  • Sources in etch but not in lenny
  • Statistics about sponsors, for packages currently in the archive
  • Open bugs per Ubuntu package

A full dump of the database, generated every two days, is also available. Be careful, it's quite big (~800 MB gzipped).
